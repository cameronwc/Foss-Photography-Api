# Pixel Helper Api

## Requirements
For this project Nodejs and npm have to be installed on your system.

## Get Started
Clone the repo

Install Dependencies
```
npm i
```

Run Server
```
node index.js
```

Browse to [localhost:3000](localhost:3000)

## Testing
This API is 100% tested with Mocha and Chai

To run these test
```
npm test
```
